Hotel Reservation System

Information

Hotel Reservation Sysytem is made using MEAN stack : MongoDB, Express JS, Angular 13 and Node JS. The app takes user reservation and stores it in the database.
There are two parts to the app:

  • Frontend, Built using Angular and Bootstrap to take user input and provide opportunity to modify or delete it, modify and delete existing reservations from the reservations table
  • Backend, built using Express JS along with Mongoose handles the incoming requests on predefined port and URLs while communicating with DB to perform necessary CRUD operations for the received request.

The app by default lands on the reservation page where user can fill in the form and create a reservation. When user clicks the Submit button, all details are collected from the form and sent the Backend Server using reservation service. If the Opertaion is successfull, an expansion panel will appear on the screen with the reservation id and other details along with an option to modify the created reservation.
